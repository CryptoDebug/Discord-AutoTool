# 🎉 Discord AutoTool - Bienvenue!

## ⚡ Démarrage Immédiat (2 minutes)

### 1️⃣ Lancer l'outil
```bash
# Windows: Double-cliquez sur → start.bat
# Linux/Mac: npm install && npm start
```

### 2️⃣ Ouvrir dans le navigateur
```
🌐 http://localhost:3000
```

### 3️⃣ Ajouter vos tokens
```
🔑 Tokens → ➕ Ajouter un Token
(copier depuis Discord Developer Mode)
```

### 4️⃣ Configurer & Démarrer
```
🚀 AutoBump ou 💬 AutoSender → Ajouter & Démarrer
```
